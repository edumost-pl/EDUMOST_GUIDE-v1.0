EduMost — двуязычная образовательная библиотека интерактивных уроков для украинских детей, обучающихся в польских школах.

Цели

  • Создать крупнейшую двуязычную библиотеку интерактивных уроков для украинских детей, обучающихся в польской школе (1–8 классы).
  • Помочь понять польскую школьную программу
  • Уменьшить языковой барьер
  • Сделать обучение интересным
  • Поддержать родителей и учителей

  1. Каждый урок решает конкретную проблему.
  2. Объяснения простым языком.
  3. PL → UA перевод терминов.
  4. Понятно без взрослого.
  5. Интерактивность обязательна.
  6. Проверка знаний в конце.
